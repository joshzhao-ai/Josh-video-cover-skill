# 风格化重绘配方（用户即梦实测·网球海报案例）

> 来源：用户在即梦网页版实测成功的人像融合案例（雪山照人物 → 美式卡通涂鸦网球海报）。
> 核心洞察：**真融合 = 人物被"以海报风格重绘"并参与画面（可换姿势/动作），而非照片抠图贴进底图。** 特征保留（发型/墨镜/衣着）但风格与全图统一 → 零"贴上去"感。

## 用户原提示词结构（模板）
```
大师级排版，极繁主义，半调图案，杂色，丝网印刷质感。点线面层次分布。
· 核心活动：<主题>
· 视觉风格：<画风>，一个男生@<人像>做<动作>，人物直接用@<人像>，风格不要变
· 信息细节：<日期/时间/地点/价格/福利等清单>、背景色彩（<色>）
```

## 移植到 CLI（dreamina image2image）
- `--images <人像>`；提示词把 @引用 改写为「第一张照片里的人物，长相五官发型衣着特征保留」。
- 人物**允许按海报风格重绘 + 摆新姿势/动作**（敲代码、跳起、持物…）——这是与"贴图式融合"的本质区别。
- 信息细节用清单式列出（即梦 5.0 能写对多行文字，网球案例 5 行全对）。

## 与两步法的分工
- **风格化重绘（本配方）**：适合插画/涂鸦/丝网印刷/复古海报类风格——人物融为画风一部分，排版自由，美感上限高。
- **两步垫图法**：适合"摄影感+真人"类参考（如小Lin说版式，用户认可其排版节奏）。
- 失败教训：底图道具+贴人两层共存（影视飓风版）容易前后层次混乱 → 这类改用本配方单步统一生成。
