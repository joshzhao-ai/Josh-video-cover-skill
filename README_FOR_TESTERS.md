# 视频封面生成 Skill · 测试版 v1.1（2026-06-10）

输入一条本地视频 → 产出 3 张达标的 3:4 竖版封面 + 可选 4:3 横版（同配方直生，5 组实测通过、无拼贴感）。由 8 条真实视频逐条打磨定版。

v1.1 更新：① 4:3 横版能力验证通过（同视觉系统直生，禁用旧"竖版改造"路径）② 配色跟品牌/主题自动取色（OpenAI→科技青蓝、Claude→珊瑚橙、复古相机→暖棕…），不再千篇一律 ③ 人像门：视频里人占比小/低清时自动隐藏"取帧"选项，引导上传照片。

## 安装

1. 把整个 `video-cover-generator-eval-20260525/` 文件夹放到 `~/.claude/skills/` 下
2. 安装并登录即梦 CLI（生图引擎，需即梦账号与积分）：
   ```bash
   curl -s https://jimeng.jianying.com/cli | bash   # 安装到 ~/.local/bin/dreamina
   dreamina user_credit                              # 触发登录并确认积分
   ```
3. 需要 `ffmpeg`（抽帧用，`brew install ffmpeg`）和 Python3

## 使用

在 Claude Code 新会话里，给视频路径并提到"封面"即可自动触发：

> 给这个视频做封面 /path/to/video.mp4

或显式点名：`/video-cover-generator-eval-20260525`

流程中它会：看帧分析 → （口播视频）问你要不要放人像（上传照片效果最佳）→ 和你确认标题（≤10字）→ 出 6 张 → 自动筛 3 张给你。对结果不满直接说哪里不行，它会把你的反馈沉淀成规则再重出。

## 能力范围（已验证的视频类型）

真人口播(AI工具/测评) · 产品测评(芯片/数码) · 实物操作(拆机/维修) · 美食纪录片 · 医学科普 · 设计教程 · 光影氛围(相机/出片) · 通用科普

## 已知限制（v1.0）

1. 标题文字偶发偏小/生僻字写错 → 已用"超采样 6 选 3"兜底，仍可能需要重出一轮
2. 取视频帧做人像 = 风格化重绘，**神似但非本人精确复刻**；要精确请上传清晰正脸照片
3. 实物操作类(手部特写)整体弱于其他类型（AI 手部细节限制）
4. 生图消耗即梦积分（约 6 张/条视频；2k 分辨率）

## 反馈

把不满意的封面截图 + 一句"哪里不行"发回来即可，规则会持续沉淀进 skill。
