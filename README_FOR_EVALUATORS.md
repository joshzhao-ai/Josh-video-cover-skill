# Video Cover Generator Skill 评测说明

Skill name: `video-cover-generator-eval-20260525`

## 你要评测什么

这个 Skill 用于把本地视频生成短视频封面，当前重点评测知识垂类：

- 科技 / AI 工具
- 健康科普
- 财经科普
- 纯口播知识内容

核心观察点不是“能不能出图”，而是：

- 标题是否有点击吸引力。
- 3:4 竖版封面是否像专业封面。
- 4:3 横版是否和竖版保持同一套视觉系统。
- 是否避免了错误垂类风格，比如睡眠视频跑出代码界面。
- 如果视频强依赖真人，是否会提示真人资产需求，而不是乱生陌生人像。

## 推荐使用方式

把整个 `.zip` 或 `.skill` 包上传到 Agent 平台。

上传后，在调试区发：

```text
使用 video-cover-generator-eval-20260525 给我上传的视频生成封面。
请先分析视频并生成标题候选，停下来让我选择标题。
```

如果平台支持本地路径，也可以说：

```text
使用 video-cover-generator-eval-20260525 给这个视频生成封面：
/path/to/video.mp4
请先停在标题候选阶段。
```

## 运行环境要求

平台或运行环境需要具备：

- Python 3
- ffmpeg
- 可执行 skill 包内 `scripts/` 脚本
- VLM 和生图 API Key

常用环境变量：

```bash
VCG_VLM_API_KEY=...
VCG_IMAGE_API_KEY=...
VCG_VLM_MODEL=...
VCG_IMAGE_MODEL=...
```

## 建议评测流程

1. 上传 1 个视频。
2. 让 Skill 先输出标题候选，不要直接全自动跑完。
3. 选择一个标题。
4. 生成 3 张 3:4 竖版封面。
5. 选择最好的一张。
6. 生成对应 4:3 横版。
7. 看最终横竖版是否一致。

## 已知限制

- 标题可能偶尔太小、贴边或字体普通，需要 refine。
- 横版可能局部元素和竖版不完全一致。
- 真人融合还在下一阶段，不建议把陌生 AI 人像视为正确结果。
- 当前最适合知识垂类 no-person 封面，不保证全品类稳定。
