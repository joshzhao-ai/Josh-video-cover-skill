# 🎬 Video Cover Generator Skill

输入一条本地视频 → AI 智能体自动产出 **3 张达标的 3:4 竖版封面**（可选 4:3 横版，同视觉系统直生）。

由 8 条真实视频（AI工具口播 / 产品测评 / 拆机维修 / 美食纪录片 / 医学科普 / 设计教程 / 光影氛围）逐条人工验收打磨定版。

## 它是怎么工作的

```
抽帧 → 智能体看帧分析(垂类/主体/钩子) → 真人口播时问你要不要放人像(上传/取帧/不放)
→ 确认标题(≤10字钩子) → 按题材配方直生 6 张(超采样) → 智能体读图按 8 条清单筛 3 张 → 交付
```

特点：
- **7 套题材配方**：真人风格化重绘(showcase级) / 产品影棚 / 实物操作 / 美食纪录片 / 医学科普 / 设计海报 / 光影氛围
- **人像门**：检测到真人口播自动停下问你；人像占比小时不给"取帧"坑你
- **排版铁律**：标题占画面 75-90%、不顶边、字体设计多样化(艺术字/书法/描边，拒绝色块平铺)
- **配色跟品牌走**：OpenAI→科技青蓝、Claude→珊瑚橙、复古相机→暖棕……不千篇一律
- **超采样+筛选**：每次多生成、智能体只把达标的给你

## 安装

### Claude Code（开箱即用）

```bash
git clone https://github.com/ciaozz2504764976-glitch/Josh-video-cover-skill.git \
  ~/.claude/skills/video-cover-generator-eval-20260525
```

新会话里给视频路径并说"做封面"即可自动触发。

### Codex / 其他智能体

把本仓库放进你的项目目录，智能体会通过 `AGENTS.md` 找到并遵循 `SKILL.md`。需要智能体支持多模态读图。

### 依赖（自行配置）

1. **即梦 dreamina CLI**（生图引擎，需自己的即梦账号与积分）：
   ```bash
   curl -s https://jimeng.jianying.com/cli | bash
   dreamina user_credit   # 触发登录
   ```
2. `ffmpeg`（抽帧）、`python3`

> 生图引擎理论上可替换：编排逻辑在 `scripts/cover_pipeline.py`，把 `dreamina` 调用换成你的文生图 API 即可（保持 3:4 / 4:3、2k）。

## 已知限制（v1.1）

- 标题文字偶发偏小/生僻字写错 → 已用超采样筛选兜底，极端情况需重出一轮
- 取视频帧做人像 = 风格化重绘，神似非精确；要精确请上传清晰正脸照片
- 实物操作类（手部特写）整体弱于其他类型
- 生图消耗即梦积分（约 6-12 张/条视频，2k 分辨率）
