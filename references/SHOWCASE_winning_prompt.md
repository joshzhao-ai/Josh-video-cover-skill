## 为什么它成（复盘 → 可复现的成因）
1. **风格化重绘，不是抠图贴图** → 人物以海报画风重绘、且"在敲代码"参与画面 → 零贴图感、光影天然一体。
2. **动作点题** → "敲代码"直接表达 AI 编程主题，主体即内容。
3. **极繁/丝网印刷/半调/点线面** 这组风格词 → 高密度但有秩序的"信息海报感"，美感上限高。
4. **文字砍到 ≤10 字** → 主标题独大、干净，符合官方铁律（v1 带3要点版偏挤，砍后立刻清爽）。
5. **珊瑚橙(Claude色)+深蓝** 配色 → 科技感且点题。

## 复现要点（沉淀给自动链路）
- 真人科技/AI 题材 + 用户传了人像 → **首选此「风格化重绘」模式**（优于两步垫图、远优于抠图贴底）。
- 动作要和主题强相关（编程→敲码；评测→持物；教学→演示）。
- 文字永远≤10字 + 小字装饰；信息别堆。
